# 🌟 Billetify - Budget Manager

**Billetify: every penny has its story** 💰  
**Billetify: cada peso tiene su historia** 🇪🇸

A beautiful, user-friendly, multi-language, multi-currency budget management application built with React.

## ✨ Features

✅ **Multi-Language Support** - Switch between English & Spanish  
✅ **Multi-Currency** - EUR, USD, COP (Colombian Peso), ZAR (South African Rand)  
✅ **Income & Expense Tracking** - Categorized transactions  
✅ **Budget Management** - Track your spending  
✅ **Beautiful Charts** - Pie charts for data visualization  
✅ **PDF Export** - Generate financial reports  
✅ **Dark Mode** - Easy on the eyes  
✅ **Local Storage** - All data saved locally on your device  
✅ **Fully Responsive** - Works on desktop, tablet, and mobile
